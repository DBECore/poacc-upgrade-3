The OpenPEPPOL BIS3 Upgrade project was started in January 2018. Project was mandated to to upgrade the PEPPOL BIS specifications so that they are aligned with the PEPPOL BIS5 invoice specification that is based on the CEN/EN 16931 eInvoicing standard.
