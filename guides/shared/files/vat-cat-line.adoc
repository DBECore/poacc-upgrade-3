
= Line VAT Category

VAT information on line level is provided in the class `cac:ClassifiedTaxCategory`.

Each line shall have the item VAT information including category code and rate. A VAT rate of zero is applied for items "Not subject to VAT" (Category E, Z, G, O, K).
