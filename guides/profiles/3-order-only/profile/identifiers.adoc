[[namespaces]]
= Namespaces

The order data model is in this PEPPOL BIS bound to the {ubl} of the order document type {ubl-order}.
The target namespace for the {ubl} Order is:

`urn:oasis:names:specification:ubl:schema:xsd:Order-2`
